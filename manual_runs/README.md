# Manual finite-grip vacancy tensile workflow

These commands are meant for running without Codex. Run only one DFT job at a time on the 32 GB machine.

All commands below assume PowerShell and the project root:

```powershell
cd C:\Users\dawso\nano_tensile_TFvW
```

## 0. Check whether a DFT job is already running

```powershell
Get-CimInstance Win32_Process |
  Where-Object { $_.Name -like 'python*' } |
  Select-Object ProcessId,ParentProcessId,CommandLine
```

If a `run_grip_tensile.py` or `prepare_grip_vacancy_wire.py` process is running, do not start another case.

## 1. Prepare geometry preview files for r = 1 to 6

This is fast and does not run DFT relaxation. It creates OVITO-checkable files under:

`cases\finite_grip_111_<r>.0nm_vacancy_tfvw\inputs_preview`

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\manual_runs\prepare_geometry_previews.ps1
```

Important: `inputs_preview` is for checking geometry only. Production tensile uses the relaxed files in `inputs`, especially `inputs\vacancy_equilibrium.vasp`, generated by the prepare command.

## 2. Run one production case

Each case does:

1. Prepare relaxed finite-grip vacancy inputs if they do not exist.
2. Run tensile cycles.
3. Stop early if the largest atomic z gap exceeds `3*d111` or if a major disconnected cluster appears.
4. Plot the latest stress-strain curve into `results\professor_review`.

Important definitions for figures and discussion:

```text
Yield strength / tensile strength:
  first peak in the literature-comparable wire stress curve,
  followed by a sustained stress drop and structural rearrangement.

Complete fracture marker:
  max atomic z gap > 3*d111, or a major disconnected cluster.

The 3*d111 rule is a conservative complete-separation marker.
It is not the strength definition used for comparison to the paper.
```

For Al [111], the default gap marker is:

```text
d111 = a0 / sqrt(3), using a0 = 4.118877004246 A
gap fracture if max atomic z gap > 3*d111 = 7.134 A
```

Run r = 1:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\manual_runs\run_one_case.ps1 -Diameter 1
```

Run r = 2:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\manual_runs\run_one_case.ps1 -Diameter 2
```

Run r = 3:


Run r = 4:

```powershell

```

Run r = 5:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\manual_runs\run_one_case.ps1 -Diameter 5
```

Run r = 6:

The r = 5 and r = 6 cases may be too large for 32 GB RAM at the current grid spacing. If Windows starts swapping heavily or the process exits, stop and run smaller cases first.

## 3. Resume an interrupted case

Use this if a run was interrupted after writing some cycles:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\manual_runs\run_one_case.ps1 -Diameter 1 -Cycles 80 -ResumeLatest
```

Replace `1` with the target case label.

## 4. Rebuild relaxed inputs from scratch

Use only when you intentionally want to overwrite the prepared `inputs` for that case:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\manual_runs\run_one_case.ps1 -Diameter 3 -ForcePrepare
```

## 5. Plot only, without running DFT

This is the command you want after a case finishes or while it is in progress.
The plot uses the vacuum-corrected cell stress as the primary curve when available:

```text
sigma_wire = sigma_cell_zz * A_cell / A_wire
```

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\manual_runs\run_one_case.ps1 -Diameter 1 -PlotOnly
```

Direct Python plotting command:

```powershell
python .\scripts\plot_grip_tensile_curve.py --case-dir .\cases\finite_grip_111_1.0nm_vacancy_tfvw --cycles-target 20 --out .\results\professor_review\r1_finite_grip_vacancy_tensile.png --copy-csv
```

For r = 2, change both `1.0nm` and `r1` to `2.0nm` and `r2`.

## 6. Run all r = 1 to 6 sequentially

Only use this when you are ready to leave the computer running for a long time:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\manual_runs\run_series_r1_to_r6.ps1
```

To start from r = 3 only:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\manual_runs\run_series_r1_to_r6.ps1 -Diameters 3,4,5,6
```

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\manual_runs\run_series_r1_to_r6.ps1 -Diameters 1,2,3,4 -Cycles 80 -ResumeLatest
```

## 7. What the workflow is doing physically

- The structure is a finite Al [111] nanowire, not a periodic z cell tensile test.
- The top and bottom grip atoms are translated as rigid bodies during loading, then fixed during relaxation.
- A single vacancy is placed in the middle free region outside the grips and close to the surface.
- Each tensile cycle stretches z by 1 percent from the current grip separation.
- Only the non-grip atoms relax after each imposed displacement.
- The publication plot reports the literature-comparable wire stress as `sigma_cell_zz * A_cell / A_wire`.
- The grip reaction stress is also plotted after subtracting the cycle-0 offset, as a mechanical cross-check.
- Yield strength is the first primary-stress peak followed by irreversible plastic rearrangement.
- Complete fracture is detected from relaxed structures using the largest z gap across the whole atomistic wire and a connectivity check.
